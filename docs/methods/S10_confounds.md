# S10: Confounds

Nuisance regressor extraction.

## Purpose

S10 extracts confound regressors for noise removal in statistical analysis.

## Confound Types

1. Motion Parameters - 6 DOF + derivatives
2. CSF Signal - Mean CSF time course
3. Physiological - Cardiac/respiratory (if available)
4. Outlier Regressors - Spike regressors for bad frames
